# SongNotes

## Introduction

SongNotes is a mern stack web tool I built for my final year project in 2024 to help songwriters capture their song ideas

## Current Features

- User account management / authentication
- Song project management
- Document Editor / Management
- Audio Recorder / Player
- Recordings / management
- Metronome function

## Future Developments

- Intelligent lyrics assistant (Rhyming Dictionary, Rhyming Scheme, Lyric suggestions)
- Chord suggestions
- Instrument Tuner
- Drum Machine
- Chord Charts (guitar)
- Tablature grid sheets (guitar)
- Music notation sheets
- Community feature to add other songwriters to a song project
- Ability to add tags and comments for other songwriters on projects, documents and recordings metadata
