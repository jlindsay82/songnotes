# SongNotes

## Introduction

SongNotes is a mern stack web tool I built for my final year project to help songwriters capture their song ideas

## Current Features

- User account management / authentication
- Song project management
- Document Editor
- Documents Management
- Audio Recorder / Player
- Recordings management
- Metronome function

## Future Developments

- Intelligent lyrics assistant (Rhyming sheme detection, Rhyme suggestions, Lyrics suggestions)
- Chord suggestor function
- Instrument Tuner
- Drum Machine
- Chord Charts (guitar)
- Tablature grid sheets (guitar)
- Music notation sheets
- Community feature to add other songwriters to a song project
- Ability to add tags and comments for other songwriters on projects, documents and recordings metadata
